# YouTube Shorts Bot

یک اسکریپت پایتون برای ساخت **شورت‌های یوتیوب** به صورت خودکار:

- دانلود ویدیو از یوتیوب
- برش ۵۰ ثانیه اول
- تبدیل به **عمودی 9:16**
- اضافه کردن **کاور تمام صفحه**
- حفظ صدای ویدیو
- آماده آپلود در یوتیوب شورت

## ⚡ پیش‌نیازها

- Python 3.8+
- pip
- ffmpeg نصب شده و در PATH
